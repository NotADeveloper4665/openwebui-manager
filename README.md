-How to Install O.W.M.

Step 1. Download the newest release and extract the zip
omm.zip -> /Downloads/owm

Step 2. Run the installer script
open a terminal and type cd Downloads/owm then run

"chmod +x install.sh" then "./install.sh"

Step 3. Verify it worked
now simply run the command openwebui-manager
If the terminal outputs "Usage:  { -start | -stop | -status | -rm }", you're good to go.

-How to use O.W.M.

Running the command openwebui-manager shows the avalible flags

Commands

| Flag | What it does |
|------|-------------|
| `-rm` | Removes openwebui from your system |
| `-start` | Starts openwebui in the background then takes its outputs and writes it to a log" |
| `-stop` | Stops openwebui in the background |
| `-status` | Shows the output of the openwebui server |
